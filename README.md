# ResearchMatch
### 🎯 [ResearchMatch: AI PhD Scholarship Tracker](https://aaqib.github.io/ResearchMatch/)
**Interactive Decision Support System for Global PhD Discovery**
- **Algorithm:** Uses a Three-Matrix Logic ($D \times A \times F$) to evaluate Desirability, Admissibility, and Feasibility.
- **Database:** Curated insights on 66 elite fellowships with real-time INR/RM currency conversion.
- **Features:** 10-question adaptive profiling, Selectivity Clusters, and Post-PhD ROI analysis.
- **Tech:** Built with Vanilla JavaScript, HTML5, and CSS3.
